SYNTAX: tadial-gradient(shape size at position, color-stop1, color-stop2,...);

The arguments of the radial-gradient() function has thr following meaning;

  1. position -Specifies the starting point of the gradient, which can be specified in units (px,em, or percentages) or keywods(lefy, bottom, etc).
  2. shape -Specifies the shape of the gradient's ending shape. It can be circle or ellipse..
  3. size -Specifies the Size of the gradient, ending shape. The default is farthest-side.
